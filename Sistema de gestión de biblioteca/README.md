# Sistema Gestión de Biblioteca

## Ejecutar el programa

Para inicar el programa debe ejecutar el siguiene comando:

```shell
python src/main.py

# O en sistemas MacOS / Linux

python3 src/main.py
```

## Estructura del Proyecto

```
README.md
src/
├── main.py
├── data/
│   └── biblioteca.py
├── models/
│   └── libro.py
└── utils/
    └── operaciones.py
```

## Descripción de Archivos

### `README.md`

Documentación principal del proyecto.

### `src/main.py`

Archivo principal de la aplicación.

### `src/data/biblioteca.py`

Módulo para la gestión de datos de la biblioteca.

### `src/models/libro.py`

Modelo de datos para representar libros.

### `src/utils/formato.py`

Utilidades para formateo de datos.

### `src/utils/operaciones.py`

Operaciones auxiliares del sistema.
